# Market

Демо-витрина интернет-магазина: **React 19**, **Vite**, **Redux Toolkit**, **React Router**, **SCSS modules**, **Feature-Sliced Design**. Бэкенда нет — данные сессии и профиля в **localStorage**.

## Демо-логины

| Email           | Пароль |
| --------------- | ------ |
| `123@gmail.com` | `123`  |
| `456@gmail.com` | `123`  |

Регистрация в демо отключена (только **Sign in**). У каждого email свои корзина, избранное и профиль в `localStorage`; при выходе сессия сбрасывается, сохранённые по email данные остаются.

## Запуск

```bash
npm install
npm run dev
```

## Функционал

- Вход / выход, защищённые маршруты (без авторизации доступны только `/login` и `/register`).
- Главная, каталог по категориям, карточка товара, недавно просмотренные.
- Корзина, оформление заказа (чекаут), избранное.
- Личный кабинет: профиль, адреса, список заказов и детали заказа.
- Статические страницы: опт, о нас, доставка, гарантия, возвраты.
- Формы с валидацией (**react-hook-form** + **zod**).
